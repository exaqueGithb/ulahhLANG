# 울랄라히힣 언어 v2.1.0

* 본 문서에서 `"문자열"`은 큰따옴표 `"`(U+0022)로 감싸인 문자들의 나열을 의미합니다.
* 본 문서에서 `"식별자"`는 한글 음절 문자 또는 영문 알파벳으로 이루어진 문자열을 의미합니다.
* 줄바꿈은 명령어의 구분에 영향을 주지 않습니다.

---

## 개요

울랄라히힣 언어는
사람이 읽을 수 있는 형태의 문장을 사용하여
출력 중심의 프로그램을 작성할 수 있도록 설계된
경량 인터프리터 기반 프로그래밍 언어입니다.

본 언어는 **웹 환경과 콘솔 환경에서 동일하게 동작**하도록 설계되었습니다.

---

## 실행 모델

* 울랄라히힣 프로그램은 **위에서 아래로 순차적으로 실행**됩니다.
* 실행 중 오류가 발생하지 않는 한, 모든 명령어는 한 번씩 실행됩니다.
* 현재 버전(v2.1.0)에서는 제어 흐름을 변경하는 명령어가 제한적으로 제공됩니다.

---

## 자료형

울랄라히힣 언어는 다음 자료형을 지원합니다.

* 문자열
* 숫자 (정수 및 실수)

모든 연산은 런타임에 동적 타입으로 처리됩니다.

---

## 출력

### `말해`

지정한 값을 표준 출력(stdout)에 출력합니다.

#### 문법

```
말해 <값>
```

#### 예시

```
말해 "Hello World"
```

출력:

```
Hello World
```

---

## 변수

### 변수 선언

변수는 `이건 나야` 키워드를 사용하여 선언합니다.

#### 문법

```
이건 나야 <식별자> = <값>
```

#### 예시

```
이건 나야 x = 10
말해 x
```

---

## 연산

기본적인 산술 연산을 지원합니다.

* `+` 덧셈
* `-` 뺄셈
* `*` 곱셈
* `/` 나눗셈

#### 예시

```
이건 나야 a = 3
이건 나야 b = 4
말해 a + b
```

---

## 반복

### `n번해`

지정한 횟수만큼 블록을 반복 실행합니다.

#### 문법

```
<n>번해 {
  <명령어>
}
```

#### 예시

```
3번해 {
  말해 "울랄라"
}
```

---

## 웹 실행

울랄라히힣 언어는 JavaScript 인터프리터로 실행됩니다.

```html
<script src="ullallahihih.js"></script>
<script>
  ullalla(`
    말해 "Hello Web"
  `)

```

---

## 오류 처리

문법 오류 또는 해석 불가능한 구문이 존재할 경우,
프로그램은 즉시 실행을 중단합니다.

#### 예시

```
말해 안녕
```

출력:

```
[울랄라 오류] 문자열은 큰따옴표로 감싸야 합니다.
```
